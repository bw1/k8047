k8047
=====

k8047 is a linux library and commando line program for the pcs10 device.
`USB ID 10cf:8047`

Installation
------------

You can install `k8047` in the usual way.

	git clone https://github.com/bw1/k8047.git
	cd k8047
	make
	k8047 -h
